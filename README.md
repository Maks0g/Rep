# Expense Tracker – Трекер расходов

Бодриков Максим
## Описание программы

Приложение для учёта личных расходов с графическим интерфейсом. Позволяет добавлять траты с указанием суммы, категории и даты, просматривать список расходов, фильтровать по категории и периоду дат, а также подсчитывать общую сумму за выбранный промежуток. Данные сохраняются в файл `expenses.json`.

## Использование

1. Убедитесь, что установлен Python 3.x.
2. Запустите программу:
   ```bash
   python expense_tracker.py
